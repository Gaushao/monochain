# hf-web

hf portal

## dev

```bash
pnpm dev:org1
# and/or
pnpm dev:org2
```

## env

`gateway/env` directory host the environment settings to organization peers, any of those may be setup on demand and can be copy to hf-net/.env file for running as default

```bash
pnpm dev:hf
```
